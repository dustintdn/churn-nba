# Data Dictionary — `customers.csv`

Synthetic SMB customer dataset generated by `src/generate_data.py` (6,000 rows, seed 42).

| Column | Type | Description |
|---|---|---|
| `customer_id` | string | Unique identifier (e.g. `SMB-00042`) |
| `tenure_months` | int | How long the customer has been subscribed (1–72 months) |
| `contract_type` | string | Billing commitment: `Month-to-month`, `Annual`, or `Two-year` |
| `plan_tier` | string | Subscription level: `Starter`, `Growth`, `Pro`, or `Enterprise` |
| `payment_method` | string | `Credit card`, `Bank transfer`, `PayPal`, or `Manual invoice` |
| `monthly_spend` | float | Monthly bill in USD, scaled by plan tier |
| `logins_per_week` | float | Average platform logins per week |
| `last_login_days` | int | Days since the customer last logged in (0–120) |
| `active_campaigns` | int | Number of advertising campaigns currently running on the platform |
| `support_tickets_90d` | int | Support tickets filed in the last 90 days |
| `discount_pct` | float | Current discount percentage on the customer's plan (0–40%) |
| `price_increase_recent` | int | Whether the customer received a recent price increase (1 = yes, 0 = no) |
| `has_account_manager` | int | Whether a dedicated account manager is assigned (1 = yes, 0 = no) |
| `nps_score` | float | Net Promoter Score (-100 to 100); ~2% of values are missing |
| `churn` | int | Target label — whether the customer churned (1 = yes, 0 = no) |
